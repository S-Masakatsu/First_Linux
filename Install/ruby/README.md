# centOSでRubyをインストールする

- 実行環境
    - CentOS Linux release 7.6.1810 (Core)

## ■ Rubyをインストール

### １. インストール

``` shell
$ yum install ruby
```

もしくは、UbuntuのようなDebianベースのディストリュビューションを使っているのであれば、

``` shell
$ apt-get install ruby
```

### ２. 確認

``` shell
$ ruby --version
ruby 2.0.0p648 (2015-12-16) [x86_64-linux]
```
