# centOSでgitをインストールする

## ■ gitをインストール

### １. インストール

``` shell
$ yum install git
```

もしくは、UbuntuのようなDebianベースのディストリュビューションを使っているのであれば、

``` shell
$ apt-get install git
```

### ２. 確認

``` shell
$ git --version
git version 1.8.3.1
```
